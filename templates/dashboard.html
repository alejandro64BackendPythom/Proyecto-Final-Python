Debes usar la siguiente API-KEY: {{ session.api_key }}

Todas las rutas se acceden a través de la base /api.

  • GET /api/productos?api_key=INSERTE API-KEY AQUI

    Descripción: Recupera listado de productos. Los parámetros page y per_page son opcionales.

    Retorno: JSON con lista de productos y metadatos de paginación.

  • PUT /api/productos/<int:id>?api_key=INSERTE API-KEY AQUI

    Descripción: Edita campos de un producto específico utilizando su ID.

    Retorno: JSON del objeto producto o 404 si no existe.

  • POST /api/productos?api_key=INSERTE API-KEY AQUI

    Descripción: Crea un nuevo producto. Requiere un cuerpo JSON con name, price y opcionalmente description.

    Ejemplo JSON: {"nombre": "Limon", "idOrigen": 2, "uMedida": "kg", "precio": 1000}

    Retorno: JSON confirmación producto mas datos ingresados (código 201).

  • DELETE /api/productos/<int:id>?api_key=INSERTE API-KEY AQUI

    Descripción: Elimina un producto específico utilizando su ID.

    Retorno: JSON de confirmación o 404 si no existe.
